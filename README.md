# This is a simple web-based tool to convert between Celsius and Fahrenheit in real-time.

Features
Convert Celsius to Fahrenheit and Fahrenheit to Celsius.
Live conversion as you input values.
Clean, responsive design with a background image.
Conversion Formulas:
Celsius to Fahrenheit:
Fahrenheit = (Celsius * 9/5) + 32

Fahrenheit to Celsius:
Celsius = (Fahrenheit - 32) * 5/9

How It Works
Enter a temperature in either field (Celsius or Fahrenheit), and the equivalent value will be instantly displayed in the other field.

How to Use
Download and open the index.html file in a browser.
Input a temperature value to get the conversion instantly!
